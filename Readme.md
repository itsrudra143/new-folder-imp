# 📘 Project Overview

**Exam Matrix** is a full-stack web-based Online Examination and Evaluation Platform designed to modernize and streamline examination processes for educational institutions and training platforms.

The platform supports:
- MCQ Questions
- Checkbox Questions
- Role-Based Authentication
- Secure Online Examination
- Real-Time Analytics
- Student & Admin Dashboards
- Calendar Scheduling
- Performance Tracking
- Fullscreen & Violation Monitoring
- Camera & Microphone Permissions
- JWT Authentication
- Class & Test Management
